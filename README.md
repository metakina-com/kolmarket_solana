# KOLMarket.ai

> **Price the Human. Empower the Agent.**

AI 驱动的 KOL 数字分身平台，让 KOL 的价值可量化、可交易、可自动化。

---

## 🎯 产品介绍

### 什么是 KOLMarket.ai？

KOLMarket.ai 是一个基于 Solana 区块链的 AI 驱动平台，为 KOL（意见领袖）提供：

- 🤖 **AI 数字分身**: 24/7 自动发推、回复粉丝、管理社区
- 💰 **收入多元化**: 打赏、订阅、交易分成、广告收入
- 📊 **影响力量化**: Mindshare 指数、数据统计、价值评估
- 🚀 **自动化运营**: Discord/Telegram Bot、自动交易、内容生成

### 核心功能

#### 对 KOL

- ✅ **解放时间**: AI 帮你处理日常运营，24/7 在线
- ✅ **增加收入**: 打赏、订阅、交易分成多种变现方式
- ✅ **扩大影响**: 永不休息的数字分身，持续与粉丝互动
- ✅ **数据洞察**: Mindshare 指数、交易数据、粉丝分析

#### 对粉丝

- ✅ **随时咨询**: 与 KOL 的 AI 分身对话，获取建议
- ✅ **跟随交易**: 订阅信号，自动跟单，不错过机会
- ✅ **实时信号**: 第一时间获取 Alpha，把握市场机会
- ✅ **打赏支持**: 使用 Solana Pay 一键打赏，支持喜欢的 KOL

## 🚀 快速开始

### 我是 KOL

1. **访问平台**: [kolmarket.ai](https://kolmarket.ai)
2. **连接钱包**: 使用 Phantom 或 Solflare 连接
3. **创建数字分身**: 在 Creator Portal 配置 Agent Suite
4. **开始赚钱**: 分享链接，粉丝可以打赏、订阅

**详细指南**: [KOL 使用手册](./docs/USER_GUIDE.md#kol-使用指南)

### 我是粉丝

1. **浏览市场**: 查看 KOL 排行榜和 Mindshare 指数
2. **与 AI 对话**: 点击 "Chat AI" 与 KOL 的数字分身对话
3. **跟随交易**: 订阅信号，在 Terminal 执行交易
4. **打赏支持**: 使用 Solana Pay 一键打赏

**详细指南**: [粉丝使用手册](./docs/USER_GUIDE.md#粉丝使用指南)

## 💡 核心优势

### 1. AI 数字分身（独特优势）

- **24/7 自动运营**: Twitter 自动发推、回复

## 💰 商业模式

### 收入来源

1. **KOL 订阅费**: $29/月（专业版）、$99/月（企业版）
2. **交易手续费**: 0.5-1% 分成
3. **打赏分成**: 平台 10%，KOL 90%
4. **广告/赞助**: CPM $5-20

### 分成模式

- **打赏**: 平台 10%，KOL 90%
- **订阅**: 平台 30%，KOL 70%
- **交易**: 平台 50%，KOL 50%
